## 2. 功能概述

优惠券管理用于维护系统中的优惠券信息。支持优惠券的新增、编辑、删除、查询，以及优惠券的启用和关闭操作。是营销活动的基础支撑模块。

## 3. 使用场景

### 场景1：运营人员新增优惠券

- **角色**：运营人员
- **前置条件**：已登录系统，拥有优惠券管理权限
- **操作步骤**：
  1. 进入优惠券管理页面
  2. 点击"新增优惠券"按钮
  3. 填写优惠券名称、面额、使用门槛、有效期等基础信息
  4. 点击保存
- **预期结果**：优惠券列表中新增一条记录，状态为"未开始"或"已启用"

### 场景2：运营人员编辑优惠券

- **角色**：运营人员
- **前置条件**：已存在优惠券数据
- **操作步骤**：
  1. 在优惠券列表中找到目标优惠券
  2. 点击"编辑"按钮
  3. 修改需要调整的字段
  4. 点击保存
- **预期结果**：修改内容保存成功，列表显示最新数据

### 场景3：运营人员启用/关闭优惠券

- **角色**：运营人员
- **前置条件**：已存在优惠券数据
- **操作步骤**：
  1. 在优惠券列表中找到目标优惠券
  2. 点击"关闭"按钮将优惠券下线
  3. 确认操作
- **预期结果**：优惠券状态变为"已关闭"，用户端不再展示该优惠券

### 场景4：运营人员删除优惠券

- **角色**：运营人员
- **前置条件**：已存在优惠券数据
- **操作步骤**：
  1. 在优惠券列表中找到目标优惠券
  2. 点击"删除"按钮
  3. 确认删除
- **预期结果**：优惠券被删除，列表刷新

## 4. 接口设计

| 接口       | 方法   | 路径                  | 说明                       |
| ---------- | ------ | --------------------- | -------------------------- |
| 优惠券列表 | GET    | /api/coupons          | 分页查询，支持名称模糊搜索 |
| 新增优惠券 | POST   | /api/coupons          | 新增优惠券                 |
| 编辑优惠券 | PUT    | /api/coupons/{id}     | 修改优惠券信息             |
| 删除优惠券 | DELETE | /api/coupons/{id}     | 删除优惠券                 |
| 启用优惠券 | PUT    | /api/coupons/{id}/on  | 启用优惠券                 |
| 关闭优惠券 | PUT    | /api/coupons/{id}/off | 关闭优惠券                 |

## 7. 业务规则

1. 优惠券名称为必填项，不可为空
2. 面额必须大于0
3. 使用门槛必须大于等于面额
4. 优惠券状态分为：已启用、已关闭
5. 新创建的优惠券默认状态为"已启用"
6. 已关闭的优惠券可以被重新启用
